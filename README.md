# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Streamlit and PDFPlumber.

## Features
- Upload PDF Resume
- Extract Name, Email and Phone
- Detect Skills
- Resume Score
- Career Recommendation
- Resume Improvement Tips
- Interview Readiness

## Technologies Used
- Python
- Streamlit
- PDFPlumber
- Regular Expressions
